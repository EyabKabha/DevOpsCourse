pipeline syntax

תגיד לי מה אתה רוצה לעשות
ואני אעשה לך אותו


properties([pipelineTriggers([cron('H 2 * * * ')])])
node("node-01"){
    stage("stageOne"){
        echo "Hello from Stage One"
        bat '''Time /t
        dir
        '''
    }
}

אם יש מילה פיפלאן - היא דיקלריטיבית
pipeline {
    agent any
    stages{
        stage("One"){
            steps{
                echo "Hello"
            }
        }
    }
}
